# Project Context

This document provides a high-level summary and general context for the ZeroRukawat project. It should be read alongside the `prd.md` and `ai_context.md`.

ZeroRukawat is an AI-powered parametric income insurance platform for delivery partners (Amazon, Flipkart). It provides a safety net by automating payouts when external disruptions (weather, strikes, local curfews) block them from working, using real-time APIs to trigger and validate the disruption.
